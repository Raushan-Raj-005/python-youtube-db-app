#  YouTube Video Manager (CLI + SQLite)

A simple command-line application built using Python and SQLite (List, Add, Update, Delete) to manage YouTube video records.
 
##  Features

*  List all videos
*  Add new videos
*  Update existing videos
*  Delete videos
*  Persistent storage using SQLite database

##  Technologies Used

* Python
* SQLite3 (built-in database)
